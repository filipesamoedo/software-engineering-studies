# C Programming – University Exercises

This folder contains C programming exercises developed as part of my university coursework in Software Engineering.

The purpose of these exercises is to practice fundamental programming concepts using the C language and strengthen problem-solving skills.

## Topics Covered

Some of the topics explored in these exercises include:

- Variables and data types
- Input and output
- Conditional statements
- Loops and control structures
- Basic algorithms
- Functions
- Arrays
- Pointers (as the course progresses)

## Purpose

These exercises are part of my continuous learning process in Software Engineering and aim to reinforce core programming fundamentals through practical implementation.
